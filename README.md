# Hotel Reservation System

## Description
A Java console-based application that allows customers to search rooms, make reservations, cancel bookings, and view booking details.

## Features
- Room Categorization
- Booking System
- Reservation Cancellation
- Payment Simulation
- File Handling
- OOP Concepts

## Technologies
- Java
- Collections
- File I/O

## Run

```bash
javac *.java
java HotelReservationSystem
```
